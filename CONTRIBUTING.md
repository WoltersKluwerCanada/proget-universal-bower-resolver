# Contributing

## Scripts

From the project root folder:

### Code quality
```
# For the main code
npm run lint
# For the test code
npm run lintTest
```

### Generate documentation
```
npm run jsDoc
```

### Generate coverage
```
npm run coverage
```
The test result will after be available in the folder `.testResult/coverage/lcov-report/index.html`.

### Unit test
```
npm run unit
```


### Run lint, unit and coverage
```
npm run test
```
